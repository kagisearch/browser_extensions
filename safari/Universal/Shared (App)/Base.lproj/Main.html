Instructions

  1. Enable extension in Safari Extension SettingsSafari Extension Settings.
  2. Enable Kagi in Private Browsing by checking the "Allow in Private Browsing" checkboxenabling the "Private Browsing" toggle switch.
  3. Go to Safari Settings -> Search and set default search engine to one you never use. We will use it to redirect searches to Kagi (because of Apple’s limitations).
  4. Select the same search engine in the dropdown here.
  5. To search in Private Browsing, set the Kagi private session link in the textbox below after copying it from your Kagi account settings.
  6. After your first search, click the Kagi extension icon in Safari toolbar and select "Always Allow on This Website".
  7. This completes the setup. You can now search with Kagi.

Sorry that Apple made this so complicated. We even started building an entire WebKit browser called Orion to make it easier to use Kagi on Mac. We’d appreciate if you gave it a try.
